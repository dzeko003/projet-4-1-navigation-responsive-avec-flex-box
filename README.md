# PROJET 4.1 — Navigation Responsive avec Flexbox

## Description

Ce projet a été réalisé dans le cadre de la formation **Akieni Academy**.

Il correspond au livrable de :

**SEMAINE 4 — CSS FLEXBOX & GRID**

L'objectif est de concevoir une **barre de navigation responsive** en utilisant **CSS Flexbox**, capable de s'adapter aux différentes tailles d'écran grâce à un **menu burger** sur mobile.

---

## Objectifs pédagogiques

À travers ce projet, les compétences suivantes ont été mises en pratique :

- Maîtriser **CSS Flexbox** pour créer des layouts en une dimension.
- Comprendre la différence entre **Flexbox** et **CSS Grid**.
- Concevoir une interface responsive.
- Utiliser les Media Queries pour adapter l'affichage.
- Créer une navigation moderne et accessible.

---

## Fonctionnalités

- Logo positionné à gauche.
- Menu de navigation responsive.
- Bouton **"Contactez-nous"** stylisé.
- Menu burger affiché sur les petits écrans.
- Animation fluide lors de l'ouverture et de la fermeture du menu.
- Mise en page réalisée entièrement avec **CSS Flexbox**.

---

## Technologies utilisées

- HTML5
- CSS3
- Flexbox
- Media Queries
- JavaScript (pour le menu burger)

---

## Responsive Design

Le projet s'adapte automatiquement aux différentes tailles d'écran grâce à des points de rupture (breakpoints), notamment à **768 px**.
